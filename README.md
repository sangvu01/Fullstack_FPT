# Fullstack Todo App

A simple fullstack Todo application built with React, Node.js, Express, MongoDB Atlas and Docker.

---

## Features

- Create Todo
- Read Todo List
- Delete Todo
- MongoDB Atlas connection
- Dockerized frontend/backend

---

## Tech Stack

### Frontend
- React
- Vite

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

### DevOps
- Docker
- Docker Compose

---

## Run Frontend

At `./frontend` folder:

```bash
npm install
npm run dev
```

Frontend:
```txt
http://localhost:5173
```

---

## Run Backend

At `./backend` folder:

```bash
npm install
node ./src/server.js
```

Backend:
```txt
http://localhost:5000
```

---

## Run With Docker

At root folder:

```bash
docker compose up --build
```
