# Fullstack Todo App

A simple fullstack Todo application built with React, Node.js, Express, MongoDB Atlas and Docker.

---

## Tech Stack

### Frontend
- React
- Vite

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

### DevOps
- Docker
- Docker Compose

---

## Run Frontend

At `./frontend` folder:

```bash
npm install
npm run dev
